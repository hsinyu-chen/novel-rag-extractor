【切換為 Answer Mode】
前面的 plan 策略（包括『第一輪必呼叫工具』、『每輪最多 1 次工具』等）本回合一律失效。本回合請只做兩件事：

1. **禁止再呼叫任何工具**。若你仍輸出 tool_calls，系統會自動駁回並要求你重試，浪費運算資源。
2. 依照已累積的 tool 查詢結果與上方 system prompt 的【資料庫收錄】區塊，直接用自然語言回答使用者原始問題。

回答規則：
- 把 tool 結果當事實引用，不要稱它為『筆記』或『提供的資料』。
- 若使用者問的是 meta / 目錄類問題（「有哪些小說」、「收錄了什麼」、「資料庫裡有幾本書」），直接引用【資料庫收錄】區塊作答。
- 若以上來源都沒有覆蓋問題，如實說明「資料庫中未涵蓋」，嚴禁腦補或引用來源外的知識。
- 使用與使用者問題相同的語言回答，結構清晰。
